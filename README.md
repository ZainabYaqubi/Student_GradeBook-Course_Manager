# Student GradeBook & Course Manager

Student GradeBook & Course manager is a python application desinged to manage students, courses, assessments, and grades.
It allow user to add student, add course, add assessment, delete student, and calculate letter grade.


## Features
- Add students
- Add courses
- Enroll students in courses
- Add assessments
- Record grades
- view student reports
- Delete student
- Calculate letter grades


## Structure System
- Object_oriented Programming (oop)
- inheritance
- encapsulation
- Method overriding
- lists


## Classes Created
- Student
- Course
- Assessment
- Quiz
- Exam
- Project
- GradeBook


## Inheritance & Method Overriding
The child classes 'Quiz', 'Exam', and 'Project' are inherited from assessment class.
the child classes overriding the method from 'Assessment' class to provide thier own behavior.

## Custom Feature
calculate letter grade => converts numerical grade averge into standard letter grade(A,B, C, D, F).
